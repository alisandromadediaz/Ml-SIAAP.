🚨 El Problema

Las señales de riesgo llegan demasiado tarde

Un porcentaje importante de estudiantes abandona o reprueba cursos porque las alertas de riesgo se detectan cuando ya no hay margen de intervención.

¿Por qué falla la detección temprana?

Los sistemas educativos tradicionales evalúan a los estudiantes al final del período, cuando ya es demasiado tarde para intervenir. Las señales de riesgo —baja asistencia, entregas tardías, poco compromiso— permanecen invisibles durante semanas.

🔴 El costo del problema: deserción escolar, reprobación masiva, recursos docentes mal distribuidos y estudiantes que podrían haberse salvado con una intervención a tiempo.

SIAAP resuelve esto aplicando Machine Learning supervisado sobre datos de comportamiento de las primeras semanas del curso, permitiendo actuar antes de que el problema sea irreversible.

🔬 Metodología

CRISP-ML(Q) — 6 Fases Completas

Cross Industry Standard Process for Machine Learning with Quality Assurance — el estándar de la industria para proyectos ML en producción.

1
Business & Data Understanding
Definir el problema educativo: predecir rendimiento (Bajo/Medio/Alto) en 3 categorías. Análisis exploratorio de 1.200 estudiantes con 10 variables de comportamiento y contexto.
2
Data Preparation
Imputación con medianas (3 columnas, ~3% faltantes), ingeniería de variables (índice de compromiso, riesgo por descanso), escalado con StandardScaler y partición 80/20 estratificada.
3
Modeling
Entrenamiento de Regresión Logística, Árbol de Decisión y Random Forest. Validación cruzada estratificada 5-folds. Optimización de hiperparámetros con GridSearchCV sobre Random Forest.
4
Evaluation
Evaluación única sobre holdout: Accuracy 0.533, F1-Macro 0.538, AUC-ROC 0.711. Análisis de importancia de variables e interpretabilidad.
5
Deployment
Empaquetado del modelo (joblib), función de recomendación pedagógica, interfaz Streamlit desplegable. Diseñado para exponer como REST API (FastAPI).
6
Monitoring & Maintenance
Sistema de alerta automática (F1 < 0.75), detección de drift con KS-test, re-entrenamiento semestral, supervisión humana docente y auditoría ética de sesgos.
🤖 Modelos

Tres Algoritmos de Aprendizaje Supervisado

Comparación rigurosa entre modelos para seleccionar el mejor candidato para predicción de rendimiento académico multiclase.

📊

Regresión Logística

Modelo base interpretable. Asume relación lineal entre variables escaladas y el log-odds de cada clase. Solver LBFGS con regularización L2. Clasificación multinomial.

F1-Macro (CV) 0.568
AUC-ROC Test ~0.710
Requiere escalado ✓ Sí
⭐ SELECCIONADO
🌲

Random Forest

Modelo ensemble con 400 árboles de decisión. Hiperparámetros optimizados con GridSearchCV. No requiere escalado. Robusto a valores atípicos y no-linealidades.

F1-Macro (CV) 0.545
AUC-ROC Test 0.711
Requiere escalado ✗ No
📈

Regresión Lineal

Predice el índice de rendimiento continuo subyacente (0-1). Permite ranking y tendencias. Complementario al modelo de clasificación para análisis de progresión académica.

R² Score ~0.47

🎓 Intervenciones

Recomendaciones Pedagógicas

El sistema no solo predice, convierte la predicción en una acción concreta para el docente.

🔴 Bajo Rendimiento

Intervención Urgente

Activar tutorías personalizadas de refuerzo semanal. Contactar al docente tutor para evaluación inmediata. Revisar carga académica y acceso a recursos.

🟡 Rendimiento Medio

Ruta Adaptativa

Asignar ruta de aprendizaje adaptativa personalizada. Monitoreo quincenal de progreso. Identificar y reforzar áreas de mejora específicas.

🟢 Alto Rendimiento

Enriquecimiento

Ofrecer contenido de profundización avanzado. Asignar rol de mentoría entre pares. Proponer proyectos de investigación y retos adicionales.
